About this application

Table shows the tasks which have been saved and sorted by due date. They can be edited and deleted


Create list of task:

  • click add task
  • fill all input in the form (due date shall be in future) and save
  • status is set to be in progress

    • Edit the task:

  • click edit this task
  • change the form as necessary and save

    • Delete the task:

  • click delete this task
  • confirm to delete the task
